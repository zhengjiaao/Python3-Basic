# Python3 pip 

pip 是 Python 包管理工具，该工具提供了对 Python 包的查找、下载、安装、卸载的功能。

目前最新的 Python 版本已经预装了 pip。

```shell
pip --version

# 下载安装包使用以下命令：
pip install some-package-name
# 移除软件包：
pip uninstall some-package-name

# 例如我们安装 numpy
pip install numpy
# 例如我们移除 numpy 包：
pip uninstall numpy

# 查看我们已经安装的软件包
pip list

```
