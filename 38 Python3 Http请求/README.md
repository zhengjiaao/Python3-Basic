# 38 Python3 Http请求

Http请求模块选型：

| 模块       | 推荐  | 描述                                                                               | 
|----------|-----|----------------------------------------------------------------------------------|
| urllib   | 推荐  | 如果你只需要简单的 HTTP 请求功能，而且希望使用 Python 标准库中的模块，可以选择使用 urllib 模块。它是 Python 内置的，无需额外安装。 |
| requests | 强推荐 | 如果你想要更简洁、方便的 API，以及更高级的功能和配置选项，建议使用 requests 库。它适合大多数的 HTTP 请求场景。                |
